Bedrock Chat

Native macOS chat client for AWS Bedrock

Download for macOS
Alpha Release — This is early software. Expect bugs, missing features, and breaking changes. Generated by an LLM, not audited, not signed, not notarized. API keys stored in plaintext.

Features

64+ Models

Claude, Llama, DeepSeek, Mistral, Qwen, Nova, and more from 16 providers

Streaming Responses

Real-time token streaming with usage display

Markdown Rendering

Rich text with code blocks and syntax highlighting

Full-Text Search

Search all your chats with Cmd+K

System Prompts

Custom system prompts per conversation

Ephemeral Mode

Chat without saving anything to disk

Context Compaction

Summarize history to save tokens

Smart Scroll

Read while streaming without being interrupted
