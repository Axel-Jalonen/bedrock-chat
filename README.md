# Bedrock Chat

This entire codebase was generated by an LLM. It has been lightly tested but treat accordingly.

A native desktop chat client for AWS Bedrock. Rust, egui, ~1.8k lines of slop.

## What it does

- 64 models across 16 providers (Anthropic, Meta, DeepSeek, Mistral, Qwen, Amazon Nova, Google, NVIDIA, etc.)
- Searchable model picker, grouped by provider
- Startup modal: paste your Bedrock API key and go, or skip for existing AWS config
- Streaming responses via ConverseStream with token usage tracking
- Markdown + code blocks via egui_commonmark
- Native macOS menu bar (File > Change API Key, New Chat, Search, Compact Context, Ephemeral Mode)
- Chat search (Cmd+K) across all conversations
- Ephemeral mode: chat without writing to disk
- Compact context: summarize conversation history to reduce token usage
- Region locked after first message (prevents mid-conversation breakage)
- System theme support (follows macOS dark/light mode)
- Subtle particle effects because why not
- Persists to SQLite at `~/.config/bedrock-chat/history.db`

## Build & run

Needs Rust 1.92+ and macOS. Get a Bedrock API key from the [AWS console](https://console.aws.amazon.com/bedrock/) and paste it on startup.

```bash
cargo run --release
```
