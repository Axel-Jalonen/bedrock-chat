# Bedrock Chat

This entire codebase was generated by an LLM in one shot. It has not been reviewed, tested with actual AWS credentials, or used by a human even once. Treat accordingly.

A native desktop chat client for AWS Bedrock. Rust, egui, ~1k lines of slop.

## What it does

- Two-pane layout (conversations left, chat right)
- Streams responses via Bedrock ConverseStream
- Markdown + code blocks via egui_commonmark
- Persists to SQLite at `~/.config/bedrock-chat/history.db`
- Model picker (Claude Sonnet 4, Sonnet 3.5 v2, Haiku 3.5, Llama 3.1 70B/8B)
- Region picker, system prompt per conversation, copy button, Ctrl+Enter to send

## Build & run

Needs Rust 1.92+ and AWS credentials configured (`~/.aws/credentials`, env vars, SSO, whatever).

```bash
cargo run --release
```

Linux: `sudo apt install libxcb-render0-dev libxcb-shape0-dev libxcb-xfixes0-dev libxkbcommon-dev libssl-dev libgtk-3-dev`
