# Bedrock Chat

[![Build](https://github.com/Axel-Jalonen/bedrock-chat/actions/workflows/rust.yml/badge.svg)](https://github.com/Axel-Jalonen/bedrock-chat/actions/workflows/rust.yml)
[![Release](https://img.shields.io/github/v/release/Axel-Jalonen/bedrock-chat?label=download)](https://github.com/Axel-Jalonen/bedrock-chat/releases/latest)

Native desktop chat client for AWS Bedrock. Rust + egui.

[**Download latest release**](https://github.com/Axel-Jalonen/bedrock-chat/releases/latest) (arm64). The binary is unsigned so macOS will complain; you know the drill (System Settings > Privacy & Security > Open Anyway).

This was generated by an LLM in a single conversation. Not audited, not signed, not notarized. Stores your API key in plaintext in `~/.config/bedrock-chat/history.db`. Use at your own risk. No liability accepted for anything.

## Features

- 64 models, 16 providers (Anthropic, Meta, DeepSeek, Mistral, Qwen, Nova, Gemma, NVIDIA, etc.)
- Searchable model picker grouped by provider
- Paste your Bedrock API key on startup and go
- Streaming via ConverseStream, token usage in the top bar
- Markdown + code blocks
- Chat search across all conversations (Cmd+K)
- Ephemeral mode (nothing written to disk)
- Compact context (summarize history to save tokens)
- Double-tap Esc to interrupt generation
- Scroll up to read while the model is still writing. It won't yank you back down.
- Follows system dark/light theme
- Particle effects

## Build from source

```bash
cargo run --release
```

Needs Rust 1.92+ and macOS.
